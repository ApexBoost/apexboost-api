Como entregar a chave ao cliente

1. Clique em Gerar para criar uma nova chave

2. Copie a chave gerada (formato: APEX-XXXX-XXXX-XXXX-XXXX)

3. Envie para o cliente pelo WhatsApp, email ou área de membros da Kiwify

4. O cliente abre o ApexBoost e digita a chave na tela de ativação

5. A chave fica vinculada ao computador do cliente — não pode ser reutilizada


⚠ Cada chave é de uso único. Se o cliente formatar o PC, precisará comprar uma nova licença.

Exceção: se quiser resetar manualmente (ex: troca de PC por garantia), use o botão "Resetar" na aba Chaves.
